Componentes reutilizados

  • • Sidebar (expanded) · Topbar con filtros globales
  • • KPI Card ×3 + KPI Card / ink ×1 (variante hero)
  • • Panel · con chart (Apex bar group)
  • • Table / default con Progress bar inline + Badge

Estados a contemplar

  • • Loading: skeleton en cada KPI + skeleton bars en chart + table-shimmer
  • • Vacío global: ilustración pequeña + CTA "crear primer cliente"
  • • Filtro sin resultados: estado vacío sólo en tabla, KPIs en cero
  • • Error: toast en topbar + reintentar por sección

Microinteracciones

  • • Hover row → preview de proyectos del cliente en sidekick
  • • Click cliente → drill-down (pantalla 2)
  • • Sparklines responden al rango global del topbar
  • • KPI ink con progreso animado al cargar (250ms ease-out)

✓ Pros: jerarquía clara, escaneo top-down, ideal pantalla principal de junta. ✗ Contras: mucha altura — el chart y la tabla quedan abajo del fold en pantallas pequeñas.

Componentes reutilizados

  • • Sidebar (collapsed) — el modo por defecto en este layout
  • • KPI band (variante nueva) — KPI Card sin borde, separados por divider vertical
  • • Table / dense con expand row + sparkline y mini-bars inline

Estados a contemplar

  • • Banda KPI con loading inline (pequeños skeletons)
  • • Tabla con group expanded/collapsed persistido por usuario
  • • Sin-resultados muestra estado vacío inline conservando filtros
  • • Error en sparkline individual no rompe la fila

Microinteracciones

  • • Click en fila cliente → expande/colapsa proyectos
  • • Click en sparkline → drill-down a Vista de Proyecto
  • • Hover en mini-bar → tooltip con monto del mes
  • • Atajo / enfoca el buscador

✓ Pros: 3× más data por pantalla. Ideal para uso operativo diario. Sparklines comunican tendencia sin ocupar un chart aparte. ✗ Contras: menos "wow" si lo presentás a cliente. Requiere familiaridad.

¿Cuál seguimos?

A · KPIs Hero

Si el dashboard se muestra a clientes / management. Lectura clara, jerarquía obvia.

C · Tabla protagonista

Si el dashboard es el cockpit de uso diario del equipo. Densidad operativa.

También se puede mezclar: banda compacta KPIs (de C) + chart hero (de B) + tabla con sparklines (de C). Decime y armo el híbrido.
